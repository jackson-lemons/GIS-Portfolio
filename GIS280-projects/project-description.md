## GIS 280 Spatial Analysis Projects

Course: GIS 280 – Introduction to GIS
Institution: North Carolina State University

This collection contains selected maps produced in ArcGIS Pro demonstrating foundational GIS skills including raster analysis, spatial modeling, and cartographic design.

## Skills Demonstrated

• Raster classification and land cover mapping
• Multi-layer thematic mapping with point datasets
• Spatial suitability analysis using geoprocessing tools
• Map layout and cartographic communication

## Project Examples
## Land Cover Classification

A raster-based land cover map created using classified land cover data. Symbology was designed to clearly distinguish major land cover categories such as forest, water, and developed areas.

## Severe Weather and Property Damage Mapping

A thematic map combining severe weather events and property damage data using layered point symbology and graduated color scales to visualize spatial patterns of damage intensity.

## Park Suitability Analysis

A spatial suitability model was developed to identify potential park locations using multiple spatial criteria. The analysis incorporated geoprocessing tools and a ModelBuilder workflow to combine datasets and generate a suitability surface.
